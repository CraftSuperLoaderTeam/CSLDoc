# 下载与运行

本文档介绍了您如何从各种渠道下载CraftSuperLoaderJava应用程序,以及如何正确启动它们

# 下载

CraftSuperLoader可以通过以下网站/链接下载到,里面有非常多的更新版本,每一个版本内会注明更新了什么功能,以及修复了上一个版本的bug

# 运行

提示

  • 对于Minecraft1.7.X-1.16.X版本,我们建议您使用JDK/JRE 1.8的运行环境
  • 对于Minecraft1.17.X-1.20.X版本,我们建议您使用JDK/JRE 17的运行环境

 

// 你的控制台
> java -Xmx2G -Dfile.encoding=UTF-8 CraftSuperLoader.jar nogui
