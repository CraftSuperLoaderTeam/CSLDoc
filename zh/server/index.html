# 关于CraftSuperLoader

# 简介

CraftSuperLoader是一个适用于Minecraft:JavaEdition多人联机的Java服务器应用程序(以下简称CSL)

CSL基于CraftBukkit服务端基础上进行开发,使其兼容大部分Bukkit插件的同时保证服务端运行的稳定性

CSL支持JavaScript、Python、Lua等多种语言编写的插件,并根据语言提供方便快捷的API

# 语言

我们支持以下编程语言来扩展开发服务端功能

语言 执行引擎 接口 备注
Java JRE/JDK CraftBukkit -
JavaScript GraalVM LLSE/CSLAPI 接口由Ethereal.和Laoxiao编写
Python GraalVM DotCS/CSLAPI 接口由7912和art提供/引擎底层交互由Mono和SuperScript编写
Lua LuaJ+JVM CSLAPI 由XingChen编写
